## 🛠️ Cómo mandarlo a la cancha (Instalación)

Para aplicar la traducción, tenés que meter mano en los archivos locales. Es un toque:

1. **Descargá** los archivos `.json` de este repositorio.
2. Buscá la carpeta del juego en **Steam**. La ruta suele ser:  
   `...\Steam\steamapps\common\Dungeon Rampage\Resources\Locale` 📂
3. **POR LAS DUDAS:** Hacé una copia de seguridad de los archivos originales (así no lloramos después). 💾
4. **Cambiá** los archivos originales por estos dos que descargaste:
   * `Locale.override.json`
   * `GameMasterLocale.override.json`
5. Mandale play y a romper todo (excepto la compu)
